# KhelGhar 🎮

KhelGhar is a fun browser-based game hub built with HTML, CSS, and JavaScript. It features simple games and logic-based interactions to engage users.

## 🌟 Features
- Multiple mini-games
- JavaScript-powered logic
- Clean and fun UI
- Responsive design
- Deployed on Vercel

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Vercel
